# Venturemond — Full Assignment Scaffold (A+B+C)

This scaffold contains a complete Next.js (App Router) + Tailwind CSS project:

- Navbar + Footer
- Hero with animations
- Services, Projects grid, About page, Contact form
- Dynamic project pages (under /projects/[slug])
- Responsive layout + Framer Motion animations
- API route for contact form

## Run locally

1. npm install
2. npm run dev
3. Open http://localhost:3000

## Deploy to Vercel

Push to GitHub → Import on Vercel → Deploy
